# xarm_description — Attribution

The robot description files (URDFs, SDFs, and meshes) in this directory were adapted from UFactory's official ROS package:

- **Upstream:** <https://github.com/xArm-Developer/xarm_ros>
- **Original package:** `xarm_description`
- **License:** BSD-3-Clause

Local modifications include conversion of selected URDFs to SDF form and additional model variants used for the experiments in the paper (e.g., `xarm7_no_hand_float.sdf`, `xarm7_3_link.sdf`). The original meshes are unchanged.
